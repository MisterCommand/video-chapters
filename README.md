## Storing chapters
Define chapters using JSON
```
[
  {
    "time": starting time in second,
    "name": string (optional)
  }
]
```
Example
```
[
  {
    "time": 0,
    "name": "Start"
  },
  {
    "time": 10
  }
]
```
